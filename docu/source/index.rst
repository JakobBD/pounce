Welcome to POUNCE's documentation!
==================================

POUNCE is a software to perform non-intrusive uncertainty quantification. 
The software is designed modular in order to guarantee flexibility in terms of solver, UQ method and system architecture. 

Supported solvers are: 
  - FLEXI

Supported UQ methods:
  - Stochastic Collocation
  - Multi Level Monte Carlo

Supported system architectures:
  - Cray Hazelhen
  - Standard Ubuntu
